# learn-github

ชลบุรี เป็นจังหวัดเก่าแก่ที่มีประวัติศาสตร์ยาวนานตั้งแต่สมัยทวารวดีและขอม โดยเดิมเป็นชุมชนประมงและเมืองท่าขนาดเล็ก (เมืองบางทราย, บางปลาสร้อย, บางละมุง) มีความเจริญรุ่งเรืองทางประวัติศาสตร์อย่างเมืองพระรถและเมืองศรีพโล ก่อนจะถูกรวมเป็นเมืองศูนย์กลางการปกครองชายฝั่งตะวันออกในสมัยรัชกาลที่ 5 และพัฒนาเป็นเมืองอุตสาหกรรม-ท่องเที่ยวหลักในปัจจุบัน
